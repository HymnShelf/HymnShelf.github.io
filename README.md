# 찬송 선반 문서

"찬송 선반"은 찬송가 악보를 등록하고 관리하며 볼 수 있는 풀스택 웹 애플리케이션입니다.
이 레포지토리는 프로젝트의 정보와 기술 명세, 그리고 개발 과정을 한글로 기록한 문서 사이트를 제공합니다.

## 문서 주요 내용

- **프로젝트 개요**: "찬송 선반" 프로젝트의 목표와 기능 설명
- **개발 과정**: 주차별 진행 상황과 회고
- **기술 명세**: 프론트엔드와 백엔드 설계
